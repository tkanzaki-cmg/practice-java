- [C1] 設問文から設問番号と設問文本体を抽出する。設問文の最後のカッコに囲まれた英字（設問形式）は無視する。
- [C2] 設問文本体を要約し、端的に聴取内容を示すタイトルを生成する。タイトル生成にあたっては、[C1]を目的を達成するための情報とし、知識ファイルをアウトプット決定にあたっての基本的な考え方として随時参照する。
 - 【例】：SC10　オンラインクレーンゲーム（ネットキャッチャー）を知っていますか。（SA）
 - ⇒生成タイトル：SC10　オンラインクレーンゲーム認知
 - 【例】:Q5　どこでも遊べてよい、について、具体的にはどのような理由から良いと思いますか。（MA）
 - ⇒生成タイトル：Q5　どこでも遊べて良い理由
- [C3] 設問文内の「／」の後にある単語を考慮し、改変せずにそのままタイトルに含める。
 - 【例】：SC14_2　あなたは現在、以下のSNSを利用していますか。最もあてはまるものをそれぞれお知らせください。／Twitter(SA)
 - ⇒生成タイトル：SC14_2　利用SNS／Twitter
 - 【例】：SC14_4　あなたは現在、以下のSNSを利用していますか。最もあてはまるものをそれぞれお知らせください。／Facebook(SA)
 - ⇒生成タイトル：SC14_4　利用SNS／Facebook
- [C4] 設問文内の特定のキーワードを検出し、それに基づくタイトルを生成する。
- [C5] 完成したタイトルを「設問番号」＋「全角スペース」＋「タイトル」の形式で出力する。「設問番号」は半角英数字である。元の設問文は出力しない。

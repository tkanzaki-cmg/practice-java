- 設問番号の先頭に通番を付与しない。
- 「／」以前の設問文が同一であれば、生成されたタイトルも同一とする。
- 「／」+「数詞」の場合は、タイトル生成時に「／」以下を無視する。
  例:「／個」、「／本」、「／歳」等の場合。
- 「自身」「家族」「お勤め」という単語がすべて設問文に含まれる場合、タイトルは必ず「除外業種」とする。
- 現住所を聴取する設問は、都道府県、市区町村いずれを聴取していても、タイトルは必ず「居住地」とする。
- 婚姻の有無を聴取する設問は、タイトルは必ず「未既婚」とする。
- 年収を聴取する設問は、「個人年収」か「世帯年収」かを判別し、必ずどちらかをタイトルとする。どちらも含まれる場合は、タイトルは必ず「個人年収、世帯年収」とする。
